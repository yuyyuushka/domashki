1. ID имеет высшую специфичность, применяется к одному элементу. Класс - меньшую специфичность, может использоваться многократно.
2. Каскад применяет стили в порядке: источник, важность, специфичность, порядок в CSS.
3. Специфичность:

ul li.important = 0,0,1,2 (1 класс + 2 элемента)
#site-title = 0,1,0,0 (1 ID)
.content > .featured-post = 0,0,2,0 (2 класса)

4. Наследуются: color, font-family, line-height. Не наследуются: margin, padding, border.
5. > выбирает только прямых потомков, а пробел любых потомков на всех уровнях вложенности.
6. :hover — стиль при наведении, :nth-child() — выбор элементов по порядку. Для интерактивности и таблиц, списков.
7. !important принудительно применяет стиль, игнорируя специфичность. Ломает естественный каскад, усложняет поддержку.
8. ::before/::after создают псевдоэлементы внутри тега. Могут содержать текст, иконки, оформление через content.
9. При одинаковой специфичности побеждает правило, которое объявлено последним в CSS.
10. Инструменты разработчика показывают примененные стили, специфичность, наследование, позволяют редактировать CSS в реальном времени.
